📈 Funnel Analysis of Google Merchandise Store

This project analyzes user drop-offs at each stage of the e-commerce funnel (Session → Product View → Add to Cart → Checkout → Purchase) using Google Analytics 4 (GA4) data. Built using Power BI for visualization, Python for transformation, and DAX for advanced calculations. The goal was to identify stages where user conversion drops significantly and suggest improvements.
